2 
3 ## About
4 This is a SPI driver to enable Texas Instruments' CC3000HostDriver to
5 work with ChibiOS/RT. This was created as part of a final year project at
6 Queen's University Belfast.
7 
8 
9 ## Setup
10 The CC3000HostDriver is not included with this repository. You can either:
11 
12 1. Run prepare.sh to get the driver and fix some issues in the CC3000HostDriver.
13 
14 or
15 
16 1. Get the latest CC3000HostDriver and place it at the root of this repository.
17 2. Run prepare.sh to then fix some issues in the CC3000HostDriver.
18 
19 See prepare.sh for more information on what it does. This is required if you
20 don't have bash and need to execute these commands manually.
21 
22 
23 ## Configuration
24 1. Copy ./config/cc3000_chibios_config.h into your project and ensure it is
25  included in the build process.
26 2. Setup your hardware SPI configuration. This should be done before calling
27  cc3000ChibiosWlanInit(). See the examples for an implementation of this
28  on a STM32 platform.
29 3. Edit cc3000_chibios_config.h to your requirements.
30 4. In your program you need to include cc3000_chibios_api.h and call
31  cc3000ChibiosWlanInit() instead of wlan_init(). See the doxygen API
32  documentation for this library for more information on its use.

35 ## Building
36 To use this, edit your project makefile (assuming the ChibiOS/RT skeleton
37 Makefile used) with the following:
38 
39  #Set the path to library
40  CC3000_CHIBIOS_DIR=<path_to_this_repo_from_your_project>
41 
42  #Include the library makefile
43  include $(CC3000_CHIBIOS_DIR)/cc3000.mk
44 
45  #Append $(CC3000SRC) to CSRC list
46  CSRC = $(PORTSRC) \ #... Existing CSRC list
47  $(CC3000SRC)
48 
49  #Append $(CC3000INC) to INCDIR list
50  INCDIR = $(PORTINC) \ #... Existing INCDIR list
51  $(CC3000INC)

54 ## Compatibility Notes
55 This has been developed against ChibiOS/RT 2.6.x running on a STM32
56 (STM32L152RC). This library has been written in such a way that any hardware
57 dependant configuration should be made externally, prior to the call to
58 cc3000ChibiosWlanInit().
59 This library depends however on HAL elements from ChibiOS/RT which may
60 not be available on all platforms. These are described below.

63 ## ChibiOS/RT Features Used
64 * HAL EXT Driver
65  * Interrupt Detect
66 * HAL SPI Driver
67  * Communications
68 * HAL PAL Driver
69  * Reading and writing to GPIO
70 * Static Thread
71  * Processing interrupts
72 * Semaphore
73  * Interrupt Signalling
74 * Mutex
75  * Permit sharing of SPI driver (optional)
